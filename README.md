# 🏥 Multi-Disease Prediction System using Machine Learning

This project presents an integrated machine learning system that predicts multiple diseases using ensemble models.  
The system is designed to improve early diagnosis and assist healthcare decision-making by analyzing patient data.

---

## 📌 Project Overview

The **Multi-Disease Prediction System** is a web-based application that predicts diseases such as:

- Diabetes  
- Lung Cancer  
- Breast Cancer  
- Parkinson’s Disease  
- Adiposity  

It uses machine learning algorithms like **Random Forest** and **Gradient Boosting** to provide accurate predictions.

The system takes patient health parameters as input and outputs the probability of different diseases.

---

## 🚀 Features

- Predicts multiple diseases in a single platform  
- Uses ensemble machine learning algorithms for higher accuracy  
- User-friendly interface for entering patient data  
- Supports early detection of chronic diseases  
- Reduces dependency on manual diagnosis

## 📊 Dataset

The datasets used in this project are collected from **Kaggle**.

They include features such as:
- Age  
- Glucose Level  
- BMI  
- Blood Pressure  
- Insulin  
- Lifestyle Factors  

---

## ⚙️ System Workflow

1. Data Collection  
2. Data Preprocessing (Cleaning, Normalization, Feature Selection)  
3. Model Training  
4. Model Evaluation  
5. Prediction Output  

---

## 📈 Results

The system achieved high accuracy in predicting diseases:

- Diabetes: **89.2%**  
- Lung Cancer: **86.4%**  
- Breast Cancer: **91.5%**  
- Adiposity: **85.3%**  
- Parkinson’s Disease: **82.7%**
